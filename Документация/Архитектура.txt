# Интерфейс Data — представляет собой выборку данных
метод Length() int — возвращает количество объектов в выборке
метод Dimension() int — возвращает размерность выборки
метод Scale(p int) Scale — возвращает шкалу, которой представлен параметр p
метод Value(a, b int) interface{} — возвращает значение параметра b объекта a

### Базовый принцип — все данные, представленные объектом Data не должны изменяться,
т. е. методы всегда возвращают одни и те же значения при тех же входных значениях.



# Интерфейс Object — представляет собой объект выборки
метод Dimension() int — возвращает количество параметров объектов
метод Scale(p int) Scale — возвращает шкалу, которой представлен параметр p
метод Value(p int) interface{} — возвращает значение параметра p



# Функции базовой обработки данных

Equal(Data, Data) bool — возвращает true, если две выборки одинаковы

Join(Data, Data) Data — сливает объекты двух выборок в одну при условии одинаковых типов шкал

RandomSelection(Data, n) (Data, Data) — случайным образом распределяет объекты исходной выборки на две
	результирующие выборки, где n — количество объектов в первой результирующей выборке

IncludeColumns(Data, int...) Data — возвращает выборку Data, где оставлены только те столбцы,
	чьи индексы перечислены как int

ExcludeColumns(Data, int...) Data — возвращает выборку Data, где имеются все столбцы исходной выборки,
	кроме тех, что перечислены как int

IncludeRange(Data, int1-int2...) Data — возвращает выборку Data, в которую включены все столбцы из
	перечисленных диапазонов. Пример: ВД(data, 4, 6, 8, 10) возвращает столбцы 4-6 и 8-10,
	ВД(data, 2) возвращает все столбцы с 2-го и до конца,
	ВД(data, 3, 6, 9) возвращает столбцы 3-6 и с 9 до конца,
	ВД(data) возвращает все столбцы.

ExcludeRange(Data, int1-int2...) Data — исключает столбцы из указанных диапазонов по аналогичному принципу.

Fix(Data) Data — создает контейнер для всех данных из исходной выборки и копирует их туда. Возвращает 
	новоссозданный контейнер. Функция необходима, когда строится большая цепочка нод.

CleanOmissions(Data) Data — очищает выборку данных от объектов, содержащих значения nil

Filter(Data, func(object Object) bool) Data — удалет из выборки объекты, для которых функция func
	возвращает false.

IsValid(Data) bool — возвращает true, если все значения соответствуют своим шкалам

...также нужны функции объединения выборок...



# Функции нормализации данных

НормализацияДесятичнымМасштабированием(Data) Data — нормализует данные методов десятичного масштабирования.

МинимакснаяНормализация(Data) Data — нормализует данные минимаксным методом.

НормализацияСтандартнымОтклонением(Data) Data — нормализует данные с помощью стандартного отклонения.

# Классификационная модель KNN
MakeClassifierKNN(d1 Data, d2 Data) ClassifierKNN — создает классификационую модель

Поля ClassifierKNN:
K int — определяет параметр K (по-умолчанию 5)
ObjectSpacingFunc — определяет функцию рассчета расстояния между объектами (по-умолчанию евклидово расстояние)

Методы ClassifierKNN:
+методы из интерфейса Model

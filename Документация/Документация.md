## Базовые функции
### Функция **AreEqual**
`func AreEqual(selection1, selection2 Data) bool` проверяет, являются ли выборки данных *selection1* и *selection2* **эквивалентными**. Две выборки называются **эквивалентными**, если каждому объекту из одной выборки соответствует **эквивалентный объект** из другой. При этом **эквивалентными объектами** называются объекты, значения параметров которых равны.

*AreEqual* возвращает false, если:
- исходные выборки *selection1* и *selection2* не являются эквивалентными
- *selection1* или *selection2* равны nil или возвращают недопустимые для методов интерфейса *Data* значения
- количество объектов в выборках, количество параметров объектов или их шкалы различаются

### Функция **AreSimilar**
`AreSimilar(selection1, selection2 Data, deviation float64) bool` проверяет, являются ли выборки данных *selection1* и *selection2* **схожими**. Две выборки называются **схожими**, если каждому объекту из одной выборки соответствует **схожий объект** из другой. При этом **схожими объектами** называются объекты, у которых номинальные значения параметров равны, а абсолютные значения не отличаются больше, чем на *deviation*.

*AreSimilar* возвращает false, если:
- исходные выборки *selection1* и *selection2* не являются схожими
- *selection1* или *selection2* равны nil или возвращают недопустимые для методов интерфейса *Data* значения
- *deviation* меньше 0 или равно NaN
- количество объектов в выборках, количество параметров объектов или их шкалы различаются

### Функция **Combine**
`func Combine(selections ...Data) Data` объединяет выборки данных *selections* в одну.

*Combine* возвращает nil, если хотя бы одна выборка из *selections*:
- равна nil или возвращает недопустимые для методов интерфейса *Data* значения
- имеет отличающиеся от других выборок параметры

### Функция **RandomSelection**
`func RandomSelection(selection Data, n int) (Data, Data)` возвращает подвыборку *n* случайных объектов *selection* в качестве первого значения и выборку всех остальных объектов в качестве второго значения. Если `n <= 0`, возвращает пустую выборку и *selection*, или если `n >= selection.Length()`, возвращает *selection* и пустую выборку. Возвращает (nil, nil), если *selection* равен nil или возвращает недопустимые для методов интерфейса *Data* значения.

### Функция **ExtractAttributes**
`func ExtractAttributes(selection Data, attributeNumbers ...int) Data` возвращает выборку, включающую атрибуты из выборки *selection*, номера которых перечислены в *attributeNumbers*.

Возвращает *nil*, если:
- выборка *selection* равна *nil*
- список номеров атрибутов *attributeNumbers* пуст
- хотя бы один из номеров *attributeNumbers* ссылается на несуществующий атрибут *selection*
